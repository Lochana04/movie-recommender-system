# 🎬 Smart Movie Recommender System

## 📌 Project Overview

This project is a content-based movie recommendation system that suggests movies based on similarity in genres, keywords, cast, and overview.

It uses Natural Language Processing (NLP) techniques and cosine similarity to recommend movies.

---

## 🚀 Features

* Content-based filtering
* Fast movie recommendations
* Simple web interface using Streamlit

---

## 🛠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit

---

## 📂 Project Files

* model.py → Recommendation logic
* app.py → Web application
* processed_movies.csv → Cleaned dataset
* movie_recommend.ipynb → Jupyter Notebook
* tmdb_5000_movies.csv → Dataset

---

## ▶️ How to Run

1. Install libraries:
   pip install pandas scikit-learn streamlit

2. Run the app:
   streamlit run app.py

---

## 📊 Output

The system recommends top 5 similar movies based on the selected movie.
